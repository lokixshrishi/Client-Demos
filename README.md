# Client Demo Websites

A collection of 6 demo websites built with React, TypeScript, Vite, and Tailwind CSS.

## Projects

| Project | Description |
|---------|-------------|
| **Client-Demo-Cafe** | Coffee shop / cafe website |
| **Client-Demo-Clinics** | Medical clinic website |
| **Client-Demo-Containers** | Container cabin / portable building services |
| **Client-Demo-HomeService** | Home services / maintenance company |
| **Client-Demo-RealEstate** | Real estate listings website |
| **Client-Demo-Restruants** | Restaurant website |

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Getting Started

Each project is independent. To run any project:

```bash
cd <project-folder>
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output will be in the `dist` folder.
